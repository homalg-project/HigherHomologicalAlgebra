  [1X2.2 [33X[0;0YExamples for systems of lifting objects[133X[101X
  
  [33X[0;0YLet  [23X(\CC,\EE)[123X  is an exact category with enough [23X\EE[123X-projective objects. For
  any   object   [23XA[123X,   there  exists  a  deflation  [23Xp_A:P_A  \to  A[123X  from  some
  [23X\EE[123X-projective  object [23XP_A[123X. We define [23X\LL[123X by the class of all [23X\EE[123X-projective
  objects  in  [23X\CC[123X.  For  an  object [23XA[123X in [23X\CC[123X, we define the the distinguished
  object  [23XL_A[123X by [23XP_A[123X and the distinguished morphism [23X\ell_A:L_A\to A[123X by [23Xp_A:P_A
  \to  A[123X.  If  [23XA[123X  happens  to  be  an  [23X\EE[123X-projective object, then [23X\ell_A[123X is a
  split-epimorphism  and  its  section  morphism [23Xs_A:A \to L_A[123X is given by the
  [23X\EE[123X-projective  lift  of  [23X\id_A[123X  along  the deflation [23X\ell_A[123X. For a morphism
  [23X\alpha:  A  \to B[123X, we define [23XL_\alpha:L_A \to L_B[123X by the [23X\EE[123X-projective lift
  of [23X\comp{\ell_A}{\alpha}[123X along the deflation [23X\ell_B[123X.[133X
  
  [33X[0;0YEvery  abelian  category  [23X\CC[123X  is exact with respect to the class [23X\EE[123X of [13Xall[113X
  short  exact  sequences  in  [23X\CC[123X.  The fact that in abelian categories every
  monomorphism is a kernel of its cokernel and every epimorphism is a cokernel
  of  its  kernel  implies that a morphism in this category is an inflation if
  and only if it is a monomorphism; and is a deflation if and only if it is an
  epimorphism.  It  also implies that an object in [23X\CC[123X is [23X\EE[123X-injective if and
  only  if  it  is  injective in the usual sense, and is [23X\EE[123X-projective if and
  only if it is projective in the usual sense.[133X

  [1X2.4-2  [33X[0;0YThe  class  of  all  objects  in  arrows  categories  represented  by[101X
  [1Xsplit-epimorphisms[133X[101X
  
  [33X[0;0YLet  [23X\CC[123X  be  an  additive  category  and  [23X\mathrm{Arr}(\CC)[123X its category of
  arrows,  i.e.,  its  objects  are  morphisms  in  [23X\CC[123X  and its morphisms are
  commutative squars in [23X\CC[123X. The class [23X\QQ[123X of all objects in [23X\mathrm{Arr}(\CC)[123X
  that  are  represented  by  split-epimorphisms defines a system of colifting
  objects  in  [23X\mathrm{Arr}(\CC)[123X.  For  an  object  [23XA=(\alpha:A_1  \to A_2)[123X in
  [23X\mathrm{Arr}(\CC)[123X,  we  define  [23XQ_A[123X by and [23Xq_A: A \to Q_A[123X by the commutative
  square whose legs are and [23X\id_{A_2}:A_2 \to A_2[123X.[133X
  
  [33X[0;0YIf  [23XA[123X  belong  to [23X\QQ[123X, then [23X\alpha:A_1 \to A_2[123X is a split-epimorphism and it
  has  a  section  morphism  is  [23X\gamma:  A_2  \to  A_1[123X. In this case [23Xq_A[123X is a
  split-monomorphism  and  its retraction morphism is given by the commutative
  square  Let  [23XB=(\beta:B_1  \to  B_2)[123X  be  an object in [23X\mathrm{Arr}(\CC)[123X and
  [23X\varphi:  A  \to  B[123X  be a morphism defined by a commutative square We define
  [23XQ_{\varphi}: Q_A \to Q_B[123X by the commutative square[133X
  
  [33X[0;0YThe  arrows  category  [23X\mathrm{Arr}(\CC)[123X  is  isomorphic  to the category of
  functors  [23X\mathrm{Hom}(1\to  2,\CC)[123X,  where [23X1 \to 2[123X is the interval category
  which  is  the  [23X\mathbb{Z}[123X-linear  closure  of  the quiver consisting of two
  vertices [23Xv_1,v_2[123X and an arrow [23Xm:v_1\to v_2[123X.[133X
  
  [33X[0;0YThe  following  operation takes and additive category [23X\CC[123X and constructs its
  arrows  category  [23X\mathrm{Arr}(\CC)[123X  and equipps it with the above system of
  colifting objects. It requires the [5XGAP[105X package[133X

  [1X2.4-4 [33X[0;0YThe class of all contractible objects in complexes categories[133X[101X
  
  [33X[0;0YLet  [23X\CC[123X  be  an  additive  category  and  [23X\Ch{\CC}[123X be its cochain complexes
  category.  An  object  [23XA[123X  in [23X\Ch{\CC}[123X is called contractible if the identity
  morphism  [23X\id_{A}[123X  is  null-homotopic.  That  is,  there  exists a family of
  morphisms     [23X(\lambda^i:A^i\to    A^{i-1})_{i\in\mathbb{Z}}[123X    such    that
  [23X\comp{\partial_{A}^i}{\lambda^{i+1}}+\comp{\lambda^i}{\partial_{A}^{i-1}}\sim
  \id_{A^i}[123X   for   all   [23Xi\in\mathbb{Z}[123X.  An  easy  verification  shows  that
  [23X\comp{\alpha}{q_B}\sim\comp{q_A}{Q_\alpha}[123X.[133X
