1 Functors

1.1 Basic functors for homotopy categories.

1.1-1 HomologyFunctor
‣ HomologyFunctor( H(A), A, n )( operation )

Returns: a functor

The first argument in the input must be the chain (resp. cochain) homotopy category of an additive category A, the second argument is A and the third argument is an integer n. The output is the n'th homology (resp. cohomology) functor :\mathrm{H}(A) \rightarrow A.
